# CPP_Robotics_Learn

本计划摒弃了C++中做网页、做界面等与你无关的内容，**完全聚焦于“面向机器人编程”**（Modern C++ for Robotics）。


---

🛠️ 学习环境准备 (第0天)
在开始之前，请配置好开发环境。推荐：
- 系统：Windows (安装WSL2 Ubuntu) 或 直接使用 Linux (Ubuntu)。无人机仿真（如Gazebo/AirSim）通常在Linux下运行最好。
- 编辑器：VS Code (安装 C++ 插件)。
- 编译器：GCC/G++.
  

---

📅 第一周：语法基础与线性思维
目标：能够写出没有语法错误的程序，理解数据如何在内存中存储。

- Day 1: Hello Drone (环境与输入输出)
  - 内容：main函数结构，#include，std::cout/cin。
  - 无人机场景：打印“System Initializing...”，输入起飞高度。
- Day 2: 变量与数据类型
  - 内容：int, double (浮点数对坐标很重要), bool, char。
  - 无人机场景：定义变量 double voltage = 12.5;, bool isArmed = false;。
- Day 3: 逻辑判断 (If/Switch)
  - 内容：条件分支。
  - 无人机场景：如果 battery < 20%，触发“返航模式”；否则“继续任务”。
- Day 4: 循环结构 (For/While)
  - 内容：循环语法，break/continue。
  - 无人机场景：模拟倒计时起飞；遍历10次传感器读数求平均值。
- Day 5: 数组与 std::vector (重点)
  - 内容：静态数组 vs 动态数组 (std::vector)。Robotics中大量使用vector存储点云或路径点。
  - 无人机场景：存储一组GPS坐标点 [(x1,y1), (x2,y2)...]。
- Day 6: 函数 (Function)
  - 内容：函数定义，参数传递，返回值。
  - 无人机场景：编写函数 calculateDistance(x1, y1, x2, y2)。
- Day 7: 周末实战 - 简易飞行状态机
  - 作业：编写一个程序，模拟无人机从“待机” -> “起飞” -> “巡航” -> “降落”的过程。用户输入指令控制状态切换。
    

---

📅 第二周：内存管理与结构化数据
目标：理解指针（C++的灵魂）和如何组织复杂数据。

- Day 8: 指针基础 (Pointers)
  - 内容：内存地址，指针变量，解引用。
  - 解释：理解为什么我们要直接操作内存（为了快）。
- Day 9: 引用 (References &)
  - 内容：引用的概念，const 引用。
  - 无人机场景：函数传递大型地图数据时，使用引用避免复制数据，节省算力。
- Day 10: 结构体 (Struct)
  - 内容：struct 定义，成员访问。
  - 无人机场景：定义 struct Waypoint { double x; double y; double z; };。
- Day 11: 字符串处理 (std::string)
  - 内容：字符串拼接、查找、比较。
  - 无人机场景：解析模拟器发送的指令字符串 "GOTO 100 200"。
- Day 12: 智能指针 (Smart Pointers)
  - 内容：std::shared_ptr, std::unique_ptr。现代C++必学，防止内存泄漏。
  - 无人机场景：管理雷达传感器对象，确保传感器断开时自动释放内存。
- Day 13: 文件读写 (File I/O)
  - 内容：fstream。
  - 无人机场景：将飞行日志（位置、高度）写入 log.txt。
- Day 14: 周末实战 - 航点加载器
  - 作业：创建一个结构体 Waypoint。从 mission.txt 文件中读取一系列坐标，存入 std::vector<Waypoint> 中，并依次打印出来。
    

---

📅 第三周：面向对象 (OOP) - 核心建模
目标：将无人机、传感器、环境抽象成“类”。这是开发仿真算法的基础。

- Day 15: 类与对象 (Class)
  - 内容：构造函数，析构函数，public/private。
  - 无人机场景：创建 class Drone，属性有坐标、电量；方法有 update(), arm()。
- Day 16: 封装与接口
  - 内容：Getters/Setters，通过方法保护数据。
  - 无人机场景：防止外部直接把电量修改为负数。
- Day 17: 继承 (Inheritance)
  - 内容：基类与派生类。
  - 无人机场景：基类 Sensor，派生出 Lidar 和 Camera。
- Day 18: 多态与虚函数 (Polymorphism)
  - 内容：virtual 关键字，重写 (Override)。
  - 无人机场景：调用 sensor->readData() 时，雷达和相机分别执行不同的读取代码。
- Day 19: 运算符重载
  - 内容：operator+, operator==。
  - 无人机场景：让两个坐标点可以直接相加 pos3 = pos1 + pos2;。
- Day 20: 命名空间与头文件管理
  - 内容：namespace, .h 与 .cpp 分离。
  - 场景：项目工程化，避免命名冲突。
- Day 21: 周末实战 - 虚拟无人机系统
  - 作业：设计一个 Drone 类，包含一个 Motor 类对象（组合）。实现无人机起飞逻辑，起飞时检查电机状态。
    

---

📅 第四周：算法入门与仿真逻辑
目标：学习 STL 容器，实现最基础的避障和路径搜索逻辑。

- Day 22: STL 容器进阶 (Map/Queue)
  - 内容：std::map (键值对), std::queue (队列)。
  - 无人机场景：用队列存储待执行的航点任务；用Map存储配置参数。
- Day 23: 算法基础 - 递归与排序
  - 内容：std::sort，简单的递归逻辑。
  - 无人机场景：对检测到的障碍物按距离进行排序。
- Day 24: 简单的避障逻辑 (人工势场法简化版)
  - 内容：向量计算（距离、方向）。
  - 算法：如果 distance(drone, obstacle) < safe_dist，则产生反向速度。
- Day 25: 栅格地图表示 (Grid Map)
  - 内容：二维数组/Vector 表示环境（0是空地，1是墙）。
  - 探索：如何用代码打印出一个 10x10 的地图。
- Day 26: 路径规划基础 (BFS 广度优先搜索)
  - 内容：图搜索算法基础。
  - 挑战：在 10x10 的网格中，找到从 (0,0) 到 (9,9) 的最短路径（不考虑障碍）。
- Day 27: 综合调试与 Lambda 表达式
  - 内容：GDB调试基础，Lambda匿名函数。
  - 场景：快速写回调函数处理传感器数据。
- Day 28-30: 月度结业项目 - 文本版“迷宫避障无人机”
  - 项目描述：
    1. 初始化一个 20x20 的字符地图（.是空地，#是障碍）。
    2. 生成一个无人机对象，初始在 (0,0)。
    3. 设置目标点 (18,18)。
    4. 核心循环：无人机每步探测周围障碍，选择离目标最近且无障碍的方向移动。
    5. 在控制台动态打印无人机的移动轨迹。

🎓 可衡量的进度标准

1. 第一周结束：你能写一个计算器，或者输入速度和时间计算飞行距离。
2. 第二周结束：你能读取一个包含坐标的文本文件，并计算出总航程。
3. 第三周结束：你能画出简单的类图，代码中有清晰的 Drone, Sensor 等类结构。
4. 第四周结束：你能写出一个在控制台里“走迷宫”的小程序。
  
🚀 下一步建议 (Month 2)

学完这一个月，你并未掌握无人机开发，但你掌握了**工具**。第二个月你需要：
1. 学习 ROS2 (Robot Operating System)：这是连接C++算法和仿真器的中间件。
2. 安装 Gazebo 或 Webots：真实的物理仿真环境。
3. 学习 Eigen 库：专门处理矩阵运算（机器人姿态解算核心）。
  
加油！这一个月会很枯燥，因为全是黑底白字的控制台，但只有打好这个基础，你才能驾驭复杂的无人机系统。
